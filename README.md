### 2. Предложите конкретные исправления для устранения этих противоречий - напишите свою версию этого фрагмента ТЗ, которая должна быть логически завершенной и непротиворечивой.

1. Пользователь может добавить в корзину от 1 до 10 единиц одного товара единовременно.  
2. Максимальное количество одного товара в корзине не может превышать 10 единиц.  
3. Пользователь может изменить количество каждого товара в диапазоне 1-10 единиц.  
4. Для удаления товара из корзины используется кнопка “удалить”.  

5. В корзине может находиться не более 5 различных товаров.  
6. Суммарное количество всех товаров в корзине не может превышать 20 единиц.  
7. При попытке добавить товар, превышающий лимиты: по количеству одного товара в корзине, количеству различных товаров или по суммарному количеству товаров, система:  
   - Отменяет операцию добавления в корзину  
   - Показывает сообщение, где сообщает причину ограничения  

8. На странице корзины отображается: список товаров, количество каждого товара, цена за единицу товара, общая стоимость позиции, общая стоимость корзины  

9. Цена на продукт фиксируется на момент добавления в корзину и больше не меняется если вдруг цена в каталоге изменилась  

10. На странице корзины должен располагаться блок рекламных товаров  

11. Рекламный блок должен отображаться только в будние дни: утром с 7:00-11:00 и вечером с 17:00 до 22:00 по часовому поясу пользователя
